3. Follow the prompts in the notebook (edit `pdb_id` to try different structures)

### Troubleshooting
- **Widgets/slider not showing:** Runtime → Restart session → Run all  
- **Package install issues:** re-run the `pip install` cell, then Run all  
- **Download issues:** try a different PDB ID (e.g., `1ubq`, `1crn`, `4hhb`)
